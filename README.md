Booki
Projet n°2 du parcours "Développeur Web" chez OpenClassrooms.

Objectif :
L'objectif était d'intégrer la maquette du site d'agence de voyage Booki en HTML et CSS. Le site doit être responsive (adapté pour mobiles, tablettes et desktop).

Technologies utilisées :
HTML5
CSS3 (Flexbox, Media Queries)
Maquette Figma
L'intégration a été réalisée à partir de la maquette Figma fournie par OpenClassrooms.

Tester le projet
Clonez le dépôt ou téléchargez les fichiers.
Ouvrez le fichier index.html dans votre navigateur préféré.
